## Intention

The idea of this project is to setup a whole all in one toolset to generate simple html sites out of markdown files.
One of the main goals is to use the project with a minimal configuration.

### Required toolset

1. Markdown parser
2. Templating engine
3. Taskrunner
4. Typescript ?
